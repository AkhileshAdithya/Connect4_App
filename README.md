# Connect4_App
A GTK frontend python app that allows you to play connect 4.

Playing Instructions - 
````
$ python Game.py
````

Prereqs - 
````
- PythonGtk thing [preinstalled on linux]
- numpy
````

Currently plays AI v/s Player
